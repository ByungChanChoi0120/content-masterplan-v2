PART 6 — 이번 주 액션

이번 주

이번 주 (4/3~4/5)

토요일이 첫 데이터 포인트입니다. 여기서 설치 완료율, 연동 비율, 막히는 지점을 확인합니다.

P0 — 토요일 설치 세션

관찰 + 데이터 수집 + 팔로업 연락선 확보

  • 참여자 옆에서 직접 설치 과정 관찰 (어디서 막히는지 기록)
  • "설치했어요? 어떻게 쓰고 있어요?" 직접 물어보기
  • 설치 완료: 업종, 어떤 업무에 쓰려는지 기록
  • 설치 미완료: 왜 안 했는지 (몰라서 / 필요 없어서 / 막혀서) 기록
  • 1:1 팔로업 연락선 확보 — 카톡 1:1 추가 또는 소그룹방 개설
  • 참여자 온도 분류 — 강함 / 보통 / 약함
  • "다음 주에 한 번 여쭤볼게요" 미리 동의 받기
P1 — 금요일 회의

실사용자 정의 + 트래킹 체계 확정

  • "실사용자" 정의 합의 (예: 설치 완료 + 1주 내 1회 이상 사용)
  • 설치 수 트래킹 방법 확정 — 대시보드 / UTM
  • 봇 레벨업 난이도와 비용 구조 논의
  • 다혜님 소재 협조 범위 확정
  • 웨비나 참가자 이메일 리스트 확보
P2 — 콘텐츠 세팅

토요일 결과 나온 후 본격 착수

  • 스티비 계정 + 발신자 인증 + 구독 폼
  • 뉴스레터 템플릿 (텍스트 중심, 이미지 2~3장)
  • 인스타 + 스레드 계정 개설
  • UTM 링크 세팅
